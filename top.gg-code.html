Moderation Commands

Command Name Description
--deleteYou can delete emojis using their name or sending them directly.
--fast-uploadYou can upload emojis fastly without giving them a name. To use this, send an emoji file with the command, send an emoji link or send the emoji itself.
--prefixYou can change the symbols (prefix) for the commands. (Uppercase prefixes / prefixes with spaces are not supported!)
--renameYou can change the name of an emoji with sending it or typing it's name.
--roleYou can limit an emoji to a certain role's users only with this command. To make the emoji usable by everyone again, type `everyone` or `remove` instead of a role name.
--uploadCreates a new emoji for your server with the file you provided.
